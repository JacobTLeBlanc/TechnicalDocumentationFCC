Hello World

We are going to throw you right onto the action! Download an IDE like Eclipse and copy paste this onto a file named HelloWorld.java

public class HelloWorld {
     public static void main(String[] args) {
        System.out.print("Hello World");
    }
}
Understanding Hello World
public class HelloWorld {
  • public: This class can be accessed by anyone, this is neccessary for the file to execute.

  • class: Declares this file as a java class

  • HelloWorld: Name of class (Must be the same as file name)

Anything inside the curly brackets is part of the class.

     public static void main(String[] args) {

This is the main method, every java program must have one. The java compiler (the thing that runs your program) looks for this exact method to run the file.


  • static: Not important for now

  • void: This method does not return a value, it executes code then terminates

  • main: This is what the compiler looks for, it the start of your program

  • String[] args is for the command-line, it is not important for a beginner.

Everything inside the curly brackets of the main method will run from top to bottom when your code is executed.

        System.out.print("Hello World");

System.out.print is used to output to the java console, it is great for debugging and outputing data. The quotation marks are used anytime we are working with words, words are called Strings in java.
